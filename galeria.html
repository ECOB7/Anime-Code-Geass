Lelouch vi Britannia

É o protagonista de Code Geass, um gênio estratégico e carismático que busca destruir o império opressor de seu pai. Usando o poder do Geass, ele manipula mentes para alcançar seu objetivo de criar um mundo melhor para sua irmã, Nunnally.

C.C.

É uma enigmática personagem de Code Geass, imortal e portadora do Geass. Ela concede esse poder a Lelouch, tornando-se sua aliada enquanto busca redenção e liberdade de sua maldição eterna.

Suzaku Kururugi

é o melhor amigo e rival de Lelouch em Code Geass. Um idealista que acredita em mudar o sistema de dentro, ele se torna o Cavaleiro de Britannia, lutando pela justiça enquanto carrega um passado marcado por sacrifícios.

Kallen Stadtfeld

Kallen Kozuki como prefere ser chamada é uma talentosa piloto de Knightmare Frames e membro da resistência em Code Geass. Dividida entre sua vida como estudante e lutadora pela liberdade, ela se torna uma das principais aliadas de Lelouch

Schniezel el Britannia

É o astuto e ambicioso primeiro-príncipe de Code Geass. Mestre em manipulação política e estratégia militar, ele é um dos principais antagonistas, rivalizando com Lelouch pelo controle do mundo.

Jeremiah Gottwald

É um dedicado oficial de Britannia em Code Geass. Conhecido por seu senso de honra, ele enfrenta dilemas enquanto navega entre lealdades e desafios inesperados.
